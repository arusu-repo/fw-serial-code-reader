# fw-serial-code-reader
フュージョンワールドのシリアルコードを読み取るアプリです
カメラでシリアルコードを読み取るWebアプリです。バックエンド不要・HTMLファイル単体で動作します。

## 対象コード形式

英数字16文字（4文字×4グループ）

```
B9XJ TM4Z AQHZ MZZ4
```

## 使い方

### 通常撮影モード
1. ブラウザで `index.html` を開く
2. カメラへのアクセスを許可する
3. シリアルコードをガイド枠に合わせる
4. 「撮影して読み取る」を押す
5. 読み取り結果を確認・修正して「保存」または「コピー」

### 連続検知モード
1. 「連続検知モード」を押す
2. コードをガイド枠に近づける（自動でスキャンを繰り返す）
3. 検知されたら「保存」を押して履歴に追加
4. 「連続検知を停止」で終了

### 結果の編集
読み取り結果は16マスのセルで表示されます。

- **タップ/クリック**：その文字を直接編集
- **← →**：前後のセルに移動
- **Backspace**：文字を消して前のセルへ
- **ペースト**：コード全体を貼り付けると各セルに自動分配

### 履歴
保存したコードは画面下部の履歴リストに蓄積されます。

- 各行の「コピー」ボタンで個別コピー
- 「全コピー」で改行区切りの一括コピー
- 「クリア」で履歴をリセット

## 動作環境

- モダンブラウザ（Chrome / Safari / Edge / Firefox）
- カメラ付きデバイス（スマートフォン・タブレット・PC）
- **HTTPS または localhost 上での提供が必要**（カメラAPIの制約）

> ローカルで開く場合は VS Code の Live Server 拡張などを使用してください。

## 技術構成

| 項目 | 内容 |
|------|------|
| OCRエンジン | [Tesseract.js v5](https://github.com/naptha/tesseract.js)（CDN読み込み） |
| 画像前処理 | グレースケール → シャープニング → Otsu二値化 |
| 認識精度向上 | 連続モードで5フレーム中3フレーム一致した結果のみ採用 |
| バックエンド | 不要（完全フロントエンド） |
| ファイル構成 | `index.html` 1ファイルのみ |
